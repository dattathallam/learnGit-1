COMPILER!

This project aims to develop an online self-sufficient educational platform which can be used to tutor students in writing a primitive compiler for an instructional programming language. The project will provide the students with a roadmap for the development process and guide them along the roadmap with supporting documentation. Following the step-by-step guidance provided by the roadmap, the students will be able to build a naive but functional compiler under minimal expert supervision.

Proceed to Step 1 to get started.
